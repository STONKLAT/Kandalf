# Kandalf
A multifunctional discord bot Which mainly focuses on music!
Multifunctional but mainly a music bot. Which has lots of features like:
Playing a song (wow)
Searching a song on youtube
Changing the volume of the song!
Music queue! (comming soon)
Looping! (also comming soon)
...and lots more!

Feel free to use any part of my code!
